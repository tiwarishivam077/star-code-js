1) Flexbox => 1D layout ==> arrange items in rows or columns

2) Items flex/expand to fill additional space or shrink to fit into smaller space

3) Flexbox is a container which contains items in a row/column ==> these items are arranged in row/column

4) In a flexbox container ==> main axis is horizontal and cross axis is vertical by default, items are arranged horizontally by default, If we change the direction of arrangement of the box the main and cross axis changes accordingly
