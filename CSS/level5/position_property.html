Relative Positions ==>> 1) Static 2) Relative 3) Absolute 4) Fixed

1) Static ===> Elements follow the normal/default document flow (top, right, bottom , left, z-index would not work)

2) Relative ===> Element's position adjusted from its Normal Position

3) Absolute ===> Position element relative to the nearest positioned ancestor

4) Fixed ===> Element positioned relative to viewport, doesnot move on scroll
