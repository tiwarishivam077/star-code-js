Block Element occupy entire space in the line i.e ==> DIV,P, H1, UL, LI they start with a new line ==> default display property is block

Inline element occupy only the space which is required i.e. span, anchor tag i.e. a, button. there is no new line b/w elements, there size can not be set easily ==> their default display property is inline
